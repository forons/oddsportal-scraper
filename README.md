# oddsportal-scraper
A scraper for OddsPortal website.

## Requirements

- `python>=3.8`
- `chromedriver` - it could be easily installed through `brew` for MacOS
- `pip install -r requirements.txt`

## Steps

1. Setup the data to scrape in `config/config.yaml`
2. Run `main.py`
